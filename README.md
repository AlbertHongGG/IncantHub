# IncantHub

IncantHub 是一個基於純物件導向 (OOP) 架構打造的專業 AI Agent 平台。系統透過嚴謹的 Schema 驅動前端動態介面，並在後端實作了企業級的 Markdown 提示詞工程與多模態圖片索引映射技術。

## 快速啟動 (Quick Start)

### 啟動後端
進入 `backend` 目錄並啟動服務：
```bash
cd backend
npm install
npm run dev
```
後端服務預設運行於 `http://localhost:3001`。

### 啟動前端
進入 `frontend` 目錄並啟動介面：
```bash
cd frontend
npm install
npm run dev
```
前端應用程式將會在瀏覽器中開啟。

## 內建 Agents 快速測試指南

為了讓您可以快速體驗系統的強大效果，以下提供各個內建 Agent 的使用範例。您可以直接複製這些範例內容貼入前端介面進行測試。

### 1. 禮貌溝通助手 (Polite Communicator)
**用途**：將粗略、情緒化或隨意的語氣，重寫為得體、專業且符合目標受眾的正式溝通內容。

* **Raw Text (原始文字) 測試範例**：
  ```text
  喂，你們那個企劃案到底什麼時候才要交啊？我都等一整天了，效率有夠差的，快點弄好給我看！
  ```
* **Target Audience (目標受眾) 測試範例**：
  ```text
  跨部門的資深主管與專案經理
  ```
* **效果預期**：AI 將保留「催促企劃案進度」的核心意圖，但會將語氣轉換為極度專業、委婉且具備職場禮儀的正式信件。

---

### 2. AI 虛擬換裝助理 (Virtual Try-On)
**用途**：多模態影像生成。透過精準的圖片索引對應 (Image Mapping)，讓 AI 分析您的主角、服裝與姿勢，生成逼真的換裝合成圖。

* **Source Character (主角圖片) - 必填 1 張**：
  請上傳一張主角的正面清晰照片。
* **Clothing Items (服裝圖片) - 必填 1~3 張**：
  請上傳衣服的商品圖（平拍或假人模特兒皆可）。
* **Target Pose (目標姿勢) - 選填 1 張**：
  （可選）上傳一張具有特定動作的參考圖，AI 會讓主角擺出該姿勢。
* **Additional Style/Prompt (附加風格/提示) 測試範例**：
  ```text
  請讓這位主角穿上提供的服裝，展現出休閒街頭風格。背景請設定在陽光明媚的巴黎街頭，確保光影自然融合，衣物材質與垂墜感必須極度逼真。
  ```
* **效果預期**：系統會在底層動態計算陣列 Index，精確教導 AI 辨識哪張圖是人、哪張是衣服。AI 將會輸出符合文字描述且完美換裝的驚豔影像。
